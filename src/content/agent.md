# Agent

## What's Agent?
Agent that executes jobs has an exclusive feature. It's No deploy and No configuration
All of job execution is done as API request from Control-bus to Agent.

## What's Agent can do?
Because Agent can be built in cloud on no configure no deploy,it can scale out easily.
