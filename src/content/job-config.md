# Job config

## Excusive execution

If you don't want to excute job multiple,turn on "Excusive execution" at jobs-&gt;\[job-name\]-&gt;setting-&gt;Execution constraints.

## Monitor execution
You can monitor execution at jobs-&gt;\[job-name\]-&gt;setting.

![image](img/notificator_dojob.png)

### Send key to notificator in response to Batch status/Exit status

Select Batch status/Exit status and select status,and input key to send to notificator at jobs-&gt;\[job-name\]-&gt;setting-&gt;notification.

### Send key to notificator in response to elapsed time

Input elapsed time,select Alert at pulldown and input key at jobs-&gt;\[job-name\]-&gt;setting-&gt;Execution constraints.

### Stop job execution in response to elapsed time

Input elapsed time and select Stop at pulldown at jobs-&gt;\[job-name\]-&gt;setting-&gt;Execution constraints.
