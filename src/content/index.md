# JobStreamer

![image](img/overview.png)

## Concept

JobStreamer is a distributed executing environment for jobs. JobStreamer can execute job that is not on the rule of JavaBatch.

[Agent](https://github.com/job-streamer/job-streamer-agent) that executes jobs has an exclusive feature.
It's No deploy and No configuration. [Management console](https://github.com/job-streamer/job-streamer-console) is front-end of [Control bus](https://github.com/job-streamer/job-streamer-control-bus).
It has a function only desplay result got from Control bus API.
