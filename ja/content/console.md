# Management Console
Management Console は Control-bus のフロントエンドで、 Control-bus の提供する API を実行し、結果を表示する機能を持ちます。
Control-bus 以上の機能は持ちません。

## Management Console でできること
- ジョブ/カレンダーの新規作成/変更/削除
- ジョブの手動実行
- agentの使用率測定
- ジョブ/カレンダーのインポート/エクスポート

などが行えます。
